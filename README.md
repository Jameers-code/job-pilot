# ✈️ Job Pilot

> Your intelligent companion for navigating the job search journey with confidence

A modern web application designed to help job seekers discover opportunities, track applications, and stay organized throughout their employment search.

## 🎯 Features

- **Smart Job Search** - Find jobs matching your skills and preferences with an intuitive search interface
- **Application Tracker** - Keep track of every application, status updates, and follow-ups in one centralized dashboard
- **Progress Insights** - Visualize your job search progress and analytics at a glance
- **User-Friendly Interface** - Clean, responsive design that works seamlessly on desktop and mobile devices

## 🚀 Quick Start

### Prerequisites

- **Node.js** 18+ 
- **npm** or **yarn** package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Jameers-code/job-pilot.git
cd job-pilot
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

### Development

Start the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

The app will automatically reload as you edit files in `app/page.tsx` or other components.

### Production Build

```bash
npm run build
npm start
# or
yarn build
yarn start
```

### Linting

Check code quality:

```bash
npm run lint
```
